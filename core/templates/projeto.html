Titulo: Sistema de agendamento para emissão de RG(SAPERG)

Descrição: Devido à demora e à dificuldade que algumas pessoas tem de tirar seu RG (principalmente as que moram em locais situados longe de postos de emissão), esse projeto tem como objetivo apresentar uma solução que reduza o tempo desse processo, por meio de um site de agendamento de atendimentos.

Equipe: Antônio Henrique / José Afonso

Tecnologias Front - JS, HTML E CSS / Back - Python(DJANGO)

Github https://github.com/ahdesenvolvimento/ahdesenvolvimento-agendamento-emissao-de-rg

Documentação: https://drive.google.com/drive/folders/1S747XuDo-9OXcpVpXReVfa19IbjR2KZb?usp=sharing

Desenvolvido por Antônio Henrique - IFPI
